General use

Getting started

Getting started with HackerPile is a quick, straightforward process.

Simply click the "Sign up" button in the toolbar and enter your info - you'll receive a confirmation email shortly after. Once you click the link in the email, you're ready to go!

Once you sign into your new account, you can navigate to the "files" page to make your first file.

Your first file

On HackerPile, the code you upload is referred to as a "file." You can create your first file by navigating to the "files" page using the link in the navigation bar.

On the "files" page, you'll notice the directory on the left side of the page - this is your pile. You can access your files in your pile, and this is where you'll create a new file or delete a file when you decide you don't want it anymore.

To create your first file, click the "Create a file" link in your pile, and specify the file name and any notes you'd like to attach to the file.

Saving files

When you edit files, there are a few ways to save them. The easiest is to wait 30 seconds! The file you're editing will be automatically saved to the cloud.

If the thought of letting the cloud do all the work is (understandably) unsettling, you can always just press the save button in the top right corner of the page.

A green save button indicates that your file is saved to the cloud, but a teal button means that your changes haven't been saved.
